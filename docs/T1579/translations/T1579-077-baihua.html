02 依住與四種所緣境

白話譯文

其次，依據法的假名安立，分別解說瑜伽所攝的止（奢摩他）與觀（毘鉢舍那）之道，應當知道，其內容如《解深密經》中所說。

慈氏菩薩對佛說：「世尊！菩薩以什麼為依、安住於什麼，才能在大乘中修習奢摩他、毘鉢舍那？」

佛告訴慈氏菩薩：「善男子！應當知道，菩薩以法的假名安立，以及不捨棄無上正等覺的願心，作為依止與安住，在大乘中修習奢摩他、毘鉢舍那。」

「世尊！如您所說，所緣境事有四種：第一，有分別影像所緣境事；第二，無分別影像所緣境事；第三，事邊際所緣境事；第四，所作成辦所緣境事。在這四種中，有幾種是奢摩他的所緣境事？有幾種是毘鉢舍那的所緣境事？有幾種是二者共同的所緣境事？」

佛說：「善男子！其中一種是奢摩他的所緣境事，就是無分別影像；其中一種是毘鉢舍那的所緣境事，就是有分別影像；其中兩種是二者共同的所緣境事，就是事邊際與所作成辦。」

03 修習止觀之方便

白話譯文

「世尊！菩薩依於止（奢摩他）與觀（毘鉢舍那）的這四種所緣境事，如何能求得止，並善於修觀？」

「善男子！如我為諸菩薩所宣說、以假名安立的法，也就是契經、應誦、記別、諷誦、自說、因緣、譬喻、本事、本生、方廣、希法、論議。菩薩對這些教法善於聽聞、善於領受，對文句善能通達流利，對義理善加尋思，對見解善能通達；便對所善思惟的法，獨自處在空閒之處，專注觀修（作意）而思惟。又對這個能思惟的心，在內心相續中作意思惟。由於這樣正確修行、反覆安住，便生起身輕安與心輕安；這就稱為止（奢摩他）。菩薩就能這樣求得止。

06 依法與不依法修止觀

白話譯文

「世尊！如經中所說，有依法修習止（奢摩他）、觀（毘鉢舍那），也有不依法修習止、觀；什麼叫依法？什麼又叫不依法？」

佛說：「善男子！如果依據所領受、所思惟的法相，而在其義理上修得奢摩他、毘鉢舍那，這就叫依法。如果不依待所領受、所思惟的一切法相，只依止他人的教誡、教授，而在其義理上修得奢摩他、毘鉢舍那，例如觀青瘀、膿爛等，或觀一切行都是無常，或觀諸行都是苦，或觀一切法都沒有我，或觀涅槃究竟寂靜；這類奢摩他、毘鉢舍那，就叫不依法。由於依止法而修得奢摩他、毘鉢舍那，所以我安立隨法行菩薩為利根性；由於不依法而修得奢摩他、毘鉢舍那，所以我安立隨信行菩薩為鈍根性。」

「世尊！如經中所說，有緣別法的奢摩他、毘鉢舍那，也有緣總法的奢摩他、毘鉢舍那；什麼叫緣別法的奢摩他、毘鉢舍那？什麼又叫緣總法的奢摩他、毘鉢舍那？」

佛說：「如果諸菩薩以各別的契經等法為所緣，依所領受、所思惟的法而修奢摩他、毘鉢舍那，這就叫緣別法的奢摩他、毘鉢舍那。如果諸菩薩就以一切契經等法為所緣，總攝為一團、一積、一分、一聚而作意思惟；思惟這一切法隨順真如、趣向真如、臨入真如，隨順菩提、隨順涅槃、隨順轉依，以及趣向彼等，或臨入彼等；又思惟這一切法宣說無量無數的善法。像這樣思惟而修奢摩他、毘鉢舍那，這就叫緣總法的奢摩他、毘鉢舍那。」

世尊說完這些話後，對慈氏菩薩說：「善哉！善哉！善男子！你現在能善於依止圓滿、最為清淨的妙瑜伽道，請問如來。你對瑜伽已獲得確定無疑、最為殊勝的善於通達（善巧）；我已為你宣說圓滿、最為清淨的妙瑜伽道。一切過去、未來正等覺者已說、當說的教法，也都如此。各位善男子、善女人，都應依此勇猛精進，正確修學。」

這時，世尊想再次宣說這個義理，便說偈頌：

在依諸法假名安立的瑜伽中，
如果修行放逸，便會失去大利益；
依止這個法與瑜伽，
若能正確修行，便能證得大覺。

若見有所獲而求脫離困難，
又認為這種見解就是得法，
慈氏啊，這樣便離瑜伽很遠，
就像大地與虛空一樣遙遠。

利益眾生的志願雖然堅固，卻不去實行；
覺悟這一點後，應勤修利益有情。
智者如此行持，經歷無量劫，
便能得到最上、遠離染污的喜悅。

若有人為了欲望而說法，
他自稱捨離欲望，卻又重新取著欲望。
愚癡的人得到無價的法寶，
反而繼續到處遊走乞討。

應捨離對爭辯、喧鬧及雜亂戲論（虛妄分別）的執著，
發起殊勝的精進。
為了救度諸天與世間眾生，
你應當修學這個瑜伽。

這時，慈氏菩薩又向佛陀稟白：「世尊！在這解深密法門中，應當怎樣稱呼這部教法？我應當如何奉持？」

佛告訴慈氏：「這名為瑜伽了義之教。對於這個瑜伽了義之教，你應當奉持。」

宣說這個瑜伽了義教時，大會中有六十萬眾生發起無上正等正覺心（阿耨多羅三藐三菩提心）；三十萬聲聞遠離塵垢（遠塵離垢），在諸法中得到清淨法眼；十五萬聲聞永盡諸漏，心得解脫；七萬五千位菩薩獲得廣大的瑜伽作意。

爾時世尊說是語已，告慈氏菩薩曰：善哉！
善哉！善男子！汝今善能依止圓滿最極清淨
妙瑜伽道、請問如來。汝於瑜伽、已得決
定最極善巧；吾已為汝宣說圓滿最極清淨
妙瑜伽道。所有一切過去未來正等覺者、已
說當說、皆亦如是。諸善男子、若善女人、皆應
依此勇猛精進，當正修學。爾時世尊欲重宣
此義、而說頌曰：
於法假立瑜伽中、若行放逸失大義；
依止此法及瑜伽、若正修行得大覺、
見有所得、求免難、若謂此見為得法；
慈氏彼去瑜伽遠、譬如大地與虛空。
利生堅固而不作，悟已、勤修利有情；
智者作此窮劫量，便得最上離染喜。
若人為欲而說法；彼名捨欲、還取欲。
愚癡得法無價寶，反更遊行而乞匃。
於諍諠雜戲論著、應捨，發起上精進。
為度諸天及世間、於此瑜伽汝當學。
爾時慈氏菩薩復白佛言：世尊！於是解深密
法門中、當何名此教？我當云何奉持？佛告
慈氏！此名瑜伽了義之教，於此瑜伽了義
之教、汝當奉持。說此瑜伽了義教時、於
大會中、有六百千眾生、發阿耨多羅三藐
三菩提心；三百千聲聞、遠塵離垢，於諸法
中得法眼淨；一百五十千聲聞、諸漏永盡，心
得解脫；七十五千菩薩、獲得廣大瑜伽作
意。
